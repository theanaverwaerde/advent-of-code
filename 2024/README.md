# Advent of Code 2024

In [Go](https://go.dev)

<https://adventofcode.com/2024>

## Prerequisites

- [Go](https://go.dev/dl/) *check with `go version`.*

## Run

Change day on the command to test code
(I use `cd` else input.txt file not found)

```bash
DAY=01 && cd 2024/$DAY/ && go run main.go && cd ../..
```
